## MAC keyboard shortcuts
- Window switching:
  - Cmd + Tab: switch apps
  - Cmd + Accent (`): switch windows of same app
  - Ctrl + Tab: switch tabs in the same window (for eg, to switch between multiple tabs in a Chrome window)
